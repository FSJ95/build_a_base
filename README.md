# build_a_base
Modulopgave 3 - IMDB datafiler

# How to build
Go to the code folder and type following command:
```
$ javac Sql.java
```

# How to run
Run following command to start the program:
```
$ java Sql
```
It is now possible to run a set of commands:
```
$ HELP
```
```
$ CREATE
```
```
$ DELETE
```
```
$ UPDATE
```
```
$ SELECT 
```
```
$ INSERT
```
```
$ EXIT
```
